# For Mac

## If connecting for the first time, in Terminal, run:
curl -o cloud-sql-proxy https://storage.googleapis.com/cloud-sql-connectors/cloud-sql-proxy/v2.8.0/cloud-sql-proxy.darwin.amd64
chmod +x cloud-sql-proxy

## For subsequent times, run starting from this code.
## In Terminal, run:
./cloud-sql-proxy vigilant-cider-474204-j7:asia-southeast1:dsa3101-storm \
